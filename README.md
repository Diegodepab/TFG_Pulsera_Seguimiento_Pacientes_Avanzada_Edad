# 📌 Pulsera de Seguimiento de Pacientes de Edad Avanzada  

Este repositorio contiene el Trabajo de Fin de Grado de **Diego De Pablo**, cuyo objetivo es el diseño e implementación de un **sistema de telemonitorización** compuesto por:  

- Prototipo de pulsera basado en un **ESP32 (LilyGO TTGO T-Display)** con sensores biométricos.  
- Plataforma web de telemedicina para la gestión de pacientes y visualización de métricas.  
- Reporte académico que documenta el proyecto.  

---

## 🛠️ Tecnologías utilizadas  

### 🔹 Hardware  
- ESP32 TTGO T-Display  
- Sensor óptico MAX30102 (frecuencia cardíaca y SpO₂)  
- Sensor MPU-6050 (acelerómetro y giroscopio para caídas y pasos)  

### 🔹 Software y plataformas  
- Arduino IDE / PlatformIO  
- Python (FastAPI, SQLAlchemy, Alembic)  
- PostgreSQL + DBeaver  
- Svelte (frontend)  
- Docker, Docker Compose, Nginx, Redis  
- MQTT (Mosquitto broker)
